  • Reporte - Hojas Impresas

    Este reporte requiere que los archivos vengan en formato *.txt (maquinas: separado por tabs, sunhive: separado por comas).

    Cómo obtener archivos

    • CP1000, Versa: desde máquina de impresión.
      • Log / [escoger rango de fechas] / Export / Current View / [Guardar como separado por tab *.txt]
    • Sunhive: Producción / Reportes / Pliegos Prensa por Orden.

  • Reporte - Vendedor

    Este reporte requiere que el archivo venga en formato *.csv (comma-separated values).

    Cómo obtener archivo

    • Sunhive: Ventas / Control de Ventas / Reporte por Cliente.
